# youtube-playlist
A demo nodejs app for youtube playlist

## Requirements

* [NodeJs](http://nodejs.org) >= 4.x 
* [mongodb](http://mongodb.org)

## Install

```sh
$ git clone git://github.com/kyue1005/youtube-playlist.git
$ npm install
```

Start app

```sh
$ npm start
```


Then visit [http://localhost:3333/](http://localhost:3333/)
